#### DRIFTUSDT 亏损复盘

**交易事实**

| 项目 | 数值 |
|------|------|
| 入场 | 0.04031 x 3721，多单，约 149.99U 名义仓位 |
| 出场 | 0.03970 x 3721，市价平仓 |
| 最大浮盈 | 约 +2.90U，最高价 0.04109 |
| 实现亏损 | 约 -2.27U 毛亏 |
| 手续费 | 约 -0.10U |
| 净结果 | 约 -2.37U |

**核心结论**

这单方向起初并非完全错误，入场后曾给到 +2U 以上浮盈；亏损主因是：**浮盈后没有保护利润，止盈挂太远，没有硬止损，仓位相对账户过大**。

**关键失误**

1. 止盈目标过远：TP1 挂 0.04120，实际最高 0.04109，差一点未成交。
2. 浮盈超过账户 20% 时未减仓，也未把止损移动到保本。
3. 没有预先挂真实止损单，最终变成人工被动止损。
4. 10U 账户使用约 150U 名义仓位，容错率极低。

**落实动作**

- 浮盈达到 +1U：至少把止损上移到接近成本。
- 浮盈达到 +1.5U 至 +2U：必须平 50% 或把全仓止损移动到保本以上。
- 第一止盈位必须覆盖“可落袋利润”，不能只挂远端目标。
- 没有硬止损单，不允许持仓超过 1 分钟。
- 单笔亏损不得再超过账户权益 8%-10%。

6. **连续止损**: 连续 2 次后当天停止交易。
